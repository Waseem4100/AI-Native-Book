# External Resources

Curated list of high-quality external resources to complement this textbook.

## Official Documentation

### ROS 2
- [ROS 2 Humble Documentation](https://docs.ros.org/en/humble/) - Official ROS 2 docs
- [ROS 2 Tutorials](https://docs.ros.org/en/humble/Tutorials.html) - Step-by-step tutorials
- [rclpy API Reference](https://docs.ros.org/en/humble/p/rclpy/) - Python client library
- [ROS Answers](https://answers.ros.org/) - Community Q&A

### Gazebo
- [Gazebo Classic Documentation](http://classic.gazebosim.org/) - Official Gazebo docs
- [Gazebo Tutorials](http://classic.gazebosim.org/tutorials) - SDF, plugins, sensors
- [gazebo_ros_pkgs](https://github.com/ros-simulation/gazebo_ros_pkgs) - ROS 2 integration

### NVIDIA Isaac
- [Isaac Sim Documentation](https://docs.omniverse.nvidia.com/isaacsim/latest/) - Official docs
- [Isaac ROS](https://nvidia-isaac-ros.github.io/) - Accelerated ROS 2 packages
- [Isaac Sim Tutorials](https://docs.omniverse.nvidia.com/isaacsim/latest/tutorials.html) - Step-by-step guides

### Navigation
- [Nav2 Documentation](https://navigation.ros.org/) - Navigation stack
- [Nav2 Tutorials](https://navigation.ros.org/tutorials/index.html) - Configuration guides

## Video Courses & Lectures

### ROS 2
- [The Construct - ROS 2 Basics](https://www.theconstructsim.com/) - Interactive ROS 2 courses
- [Articulated Robotics (YouTube)](https://www.youtube.com/@ArticulatedRobotics) - ROS 2 tutorials

### Robotics Fundamentals
- [Modern Robotics (Northwestern)](https://modernrobotics.northwestern.edu/) - Free textbook + videos
- [MIT OpenCourseWare - Robotics](https://ocw.mit.edu/courses/electrical-engineering-and-computer-science/) - University lectures

## Books

### Robotics & ROS
- *Programming Robots with ROS* by Morgan Quigley et al. (O'Reilly, 2015)
- *A Gentle Introduction to ROS* by Jason M. O'Kane (2013, free PDF)
- *Modern Robotics: Mechanics, Planning, and Control* by Lynch & Park

### AI & Machine Learning
- *Artificial Intelligence: A Modern Approach* by Russell & Norvig
- *Deep Learning* by Goodfellow, Bengio, Courville (free online)

## Tools & Utilities

### Visualization
- [RViz2](https://github.com/ros2/rviz) - 3D visualization for ROS 2
- [Foxglove Studio](https://foxglove.dev/) - Modern robotics visualization
- [PlotJuggler](https://www.plotjuggler.io/) - Time-series data plotting

### Simulation
- [Webots](https://cyberbotics.com/) - Alternative robot simulator
- [PyBullet](https://pybullet.org/) - Lightweight physics simulation
- [MuJoCo](https://mujoco.org/) - Advanced physics engine

### Development
- [colcon](https://colcon.readthedocs.io/) - Build tool for ROS 2
- [rosdep](http://docs.ros.org/en/independent/api/rosdep/) - Dependency management
